# Portfolio Room 3D

Portfolio interactivo en 3D desarrollado como proyecto final de la asignatura Informática Gráfica (4º año, Grado en Ingeniería Informática). Una alternativa innovadora a los portfolios web tradicionales que permite explorar un espacio tridimensional inmersivo.

## Descripción

Portfolio Room 3D es una aplicación web interactiva que presenta un portfolio personal en formato de habitación 3D explorable. El proyecto demuestra la aplicación práctica de conceptos de gráficos por computador mediante Three.js, permitiendo a los visitantes navegar por un espacio virtual que contiene información profesional, proyectos y enlaces de contacto.

## Tecnologías

Este proyecto ha sido desarrollado utilizando las siguientes tecnologías:

- **JavaScript** - Lenguaje de programación principal
- **Three.js** - Librería para renderizado de gráficos 3D en WebGL
- **Vite** - Herramienta de desarrollo y empaquetado rápido
- **Navegador web** - Plataforma de ejecución (compatible con navegadores modernos)

## Funcionalidades

El proyecto incluye las siguientes características:

- **Escena 3D interactiva** - Habitación completamente modelada en 3D que puede ser explorada libremente
- **Controles orbitales** - Sistema de navegación intuitivo que permite rotar, hacer zoom y desplazarse por la escena
- **Elementos interactivos** - Objetos dentro de la habitación que permiten acceder a:
  - Demos de proyectos realizados
  - Currículum vitae (CV)
  - Enlaces a redes sociales y perfiles profesionales

## Instalación y uso

### Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Node.js (versión 14 o superior)
- npm (incluido con Node.js)
- Un navegador web moderno (Chrome, Firefox, Edge, Safari)

### Pasos para ejecutar en local

1. **Clonar el repositorio**
```bash
git clone https://github.com/juanfradelrm/portfolio-room-3d.git
cd portfolio-room-3d
```

2. **Instalar dependencias**
```bash
npm install
```

3. **Ejecutar en modo desarrollo**
```bash
npm run dev
```

4. **Acceder a la aplicación**

Abre tu navegador y visita la URL que Vite proporciona en la consola (normalmente `http://localhost:5173`)
